# Adventure Advisors

## General Info
This application is designed to help you plan out a last minute roadtrip, providing current weather and traffic based on a users specific location inquery. 

## Technologies Used
- CSS
- HTML
- Bootstrap
- Weather API
- Goole Maps API
- Firebase
